PEGBIS (Python Efficient Graph-Based Image Segmentation)

Python implementation of "Efficient Graph-Based Image Segmentation" paper written by P. Felzenszwalb, D. Huttenlocher. The paper is available: http://cs.brown.edu/~pff/papers/seg-ijcv.pdf
C++ implementation is written by the author and is available on: http://cs.brown.edu/~pff/segment/
Python implementation is written by Ghassem Alaee and is available on: https://github.com/salaee/pegbis

Requirements

Python 3.5

Libraries used:

scipy and matplotlib
